=========================



### Installation
```
$ npm install
```

### Develop
```
$ npm run dev
```
And open in browser: [http://localhost:3000](http://localhost:3000)

### Run app
```
$ npm run build
$ npm start
```
And open in browser: [http://localhost:3000](http://localhost:3000)

### Run test
```
$ npm test
```

### Deploy
https://leancloud.cn/cloud.html?appid=d4s2Cn23b1rcgs2AeD9PQdEM-gzGzoHsz#/deploy
